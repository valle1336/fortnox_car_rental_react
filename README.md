Fortnox Car Rental – Frontend

React SPA for the Fortnox Car Rental assignment.

Requirements:
Node.js (v18+ rekommenderas)
NPM

1. Clone the repo
2. Environment config (Create your own .env file and add "REACT_APP_API_URL=someApiUrl"
3. npm install
4. Start the app (npm start)

Routes: 
/rent page for users to access rent form for renting cars
/admin overview table of all rents made
/ Index page

Testing: 
API and component testing,
Run tests (npm test)

Tech stack: 
1. React (19.x)
2. React Router
3. Fetch API
4. Plain CSS (inline)
5. Jest + Testing Library

Notes: 
All validation is handled by REST-API (If not, also validated by frontend)
